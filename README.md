# CSVParser

A simple online tool to view csv files in tabular format. Can be used for data analysis, Or ***when MS-Excel doesn't open :stuck_out_tongue_winking_eye:***

# Features / Demo

- ***Generate table***
- ***Detects Duplicates***
- ***Remove Duplicates***
- ***Sorting based on Columns***
- ***Column based filtering***
- ***Add and Delete Rows***
- ***Edit Rows data***
- ***Download (.Csv)***
