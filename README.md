# CSVParser

A simple online tool to view csv files in tabular format. Can be used for data analysis, Or ***when MS-Excel doesn't open :stuck_out_tongue_winking_eye:***

# How It works

- generate table
- detects Duplicates
- Remove Duplicates
- Sorting
